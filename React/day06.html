# axios

  1. 纯粹 只做前后台数据交互
  2. 基于 Promise

# 1.使用：

  1. axios()
  2. axios.get() axios.post()

# 2.请求方式：

  1. get(url，config) 获取
  2. post(url,data,config) 新增
  3. put(url,data,config) 全局更新
  4. patch(url,data,config) 局部更新
  5. delete(url,config) 删除

# 3.config 配置对象

  1. method
  2. url
  3. baseURL
  4. data
  5. params
  6. headers
  7. timeout

# 4. 安装使用

    npm i axios --force
    import axios from "axios"
