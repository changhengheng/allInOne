# 前言

英语有很多音标体系。

英美音标对比

# 各种音标体系

# 国际音标（IPA）

简称IPA(International Phonetic Alphabet)，是100多年前被一群语言学家发明出来，用来给世界上所有语言注音的的音标符号。

# 英语音标（DJ音标）

我们平时所见的英语音标，来自英国教授丹尼尔·琼斯教授。最早是借鉴于国际音标（IPA）用来给英语注音的，这套音标体系传到中国后被以讹传讹称作了“国际音标”，但真正的国际音标（IPA）要复杂的多。为了避免混淆，琼斯教授的这套音标体系一般称作“DJ音标”。

# IPA63和IPA88

IPA63是琼斯教授英语发音词典1963年版本的DJ音标，多年前还在学校里被当做标准教授。

IPA88是现今常见的DJ音标新版本，它是1988年的版本，是由琼斯教授的学生A.C.Gimson修订的，所以也被称作Ginson音标。

# KK音标

美国研学家John Kenyon 和Thomas Knott 编撰了一本“美国英语发音词典”，其中大部分注音符号和DJ音标一样也是来自国际音标，被称作KK音标，在台湾地区被广泛使用（基本只在台湾地区使用）。

# 英语母语人士使用哪种音标体系？

英语母语人士往往会采取所谓respelling的方法，就是用常用词里的音去标注，而基本上不使用音标。

# 关于不同音标体系的三个常见问题

# 1. 为什么要知道有多种不同的音标？

并不需要了解多套音标，而是为了避免对不同的音标符号产生疑问。

# 2. 学哪一套音标好？

选一套学就行。这套视频讲的是DJ音标，因为这套音标被国内广泛使用。

# 3. 学美式英语，是否应该选KK音标？

DJ音标也是可以用来给美式英语注音的，所以使用DJ音标还是KK音标都是可以的。

# 人类发音过程

可以粗略理解为是肺部气流被挤压，一路上行，来到喉咙的“声门”。这就是发音的“动力”。声门其实是两瓣声带的开口，当气流从肺部冲出声门时，咱们可以选择让气流振动声带，再让气流继续上行。如果是这样由声带振动作为音源而发出的音，称之为“浊音”（voiced sound）。当然，也可以选择在气流通过声门时不振动声带，直接让气流通过，这样发出的音，称之为“清音”（voiceless sound）。气流离开声门，来到口腔和鼻腔处，咱们的唇舌等器官就可以开始参与发音了。嘴唇可以通过张开和闭合来控制气流通行，舌头可以通过抬高和降低改变气流的通行。舌头还可以和口腔不同的部位（比如齿、齿龈、硬腭）接触，阻碍气流。而软腭、小舌也可以参与发音。另外，咱们还能选择是否让气流通过鼻腔。可想而知，这样不同的发音方式的组合可以很复杂。

// 声带振动发出的音叫“浊音”，声带不振动发出的音叫“清音”

除了“清音”和“浊音”这种分类，咱们还可以按发音过程中，气流是否受到阻碍，把人类能发出的音归为两大类。一类是发音过程中气流通过口腔时不受阻碍而发出的音，称之为“元音”（vowels），也有的语音体系称之为“母音”。注意，这里说的是“不受阻碍”，而舌头还是可以通过抬高和降低，也就是改变舌头的“形状”参与发音的。只不过，**舌头不与任何口腔部位接触，否则就阻碍气流了。当然，嘴唇也不能闭拢，否则也形成阻碍，发出的就不是元音了。**剩下的一类，就是在发音过程中，气流在全部时间或一部分时间被阻碍，所发出的音。称之为“辅音”（consonants），也有体系称之为“子音”。

// 发音过程中气流通过口腔时不受阻碍而发出的音叫元音，气流受阻碍发出的音叫辅音

最后，气流离开人体，发“一个音”的过程也就结束了。之后就可以开始下一个音的发音过程。

# 元音辅音分类方式和清音浊音分类方式的联系

理论上来说，元音可以有清浊，即“浊元音”、“清元音”，辅音也可以有“清辅音”、“浊辅音”。但是清元音非常少见，而绝大多数情况下，元音的发音都是有声带振动的，所以一般粗略认为元音都是浊音。而辅音中的浊辅音和清辅音则有很明显的对立。

// 元音一般都是浊音，而辅音有清辅音和浊辅音。

# 元音三要素

发元音时可控的因素并不多，其实只有舌头和嘴唇能够通过有限的调整来发出不同的元音。具体来说，影响元音的三个要素是：圆唇度、舌位高低、舌位前后。

# 圆唇度

嘴唇在不闭合的情况下（闭合时会阻碍气流，发出的就是辅音），可以圆唇（rounded）也可以不圆唇（unrounded），称这个嘴唇圆的程度为“圆唇度”（roundness）

比如汉语“离”和“驴”这两个字的韵母，区别只是圆唇度不同，而舌头的位置是一样的，自己感受一下。

# 舌位高低

舌头要么抬高降低，要么前伸后缩。发音时，舌头并不总是平的，所以我们以舌头在发音时隆起的最高点来做参照，可以称这个舌头在口腔中的最高点为“舌位”。舌头抬高，舌位就高；舌头降低，舌位就低。

这是纵向的舌位，也就是舌位的高低（height）。比如：离和驴的韵母发音时，舌头和上颚（分软腭和硬腭）非常接近，所以舌位非常高，自己感受一下，是不是舌头很接近上颚，但并不碰到上颚？这就是舌位纵向的“高低”。

# 舌位的前后

舌位的横向自然就是前后（backness）。再比较普通话“驴”的韵母和“炉”的韵母。前者是个圆唇，设为很前的元音（拼音ǘ），后者同样圆唇，但是个舌位很后的元音（拼音ú）。感受一下，是不是在发后者的元音时，舌头后缩了？这就是舌位横向的“前后”。

# 其他补充

以上是影响元音发音的三要素，只有这三个，没有其他的了

# 张嘴大小

嘴巴张开是大是小是由舌位决定的，归根结底是由舌位的高低前后决定的。

# 元音图

元音图（vowel diagram或vowel chart）直观地以视觉的方式表现了影响元音发音的三要素：圆唇度、舌位高低、舌位前后，便于进行学习和纠音。

把元音发音时，感知到的舌位的高低和前后标注在一个上大下小的元音四边形（vowel quadrilateral）内。舌位高而前的元音标注在这个四边形的左上方，舌位高而后的元音标注在这个四边形的右上方，舌位低而前的元音标注在这个四边形的左下方，舌位低而后的元音标注在这个四边形的右下方。而这个四边形之所以上大下小是因为高元音舌位横向（前后）的差别要大于低元音舌位横向（前后）的差别

//仔细看图，学音标的时候脑子里要时刻都有这个图

舌位的高低和前后这两个原因要素可以很直观地标注。而圆唇度只能依照惯例，把同一个舌位上的不圆唇的元音标注在前，而把圆唇的元音标注在后。

注意：口腔并不是真的长得和一个四边形一样，所以元音图上各个位置是咱们所“感知到的”元音舌位，是抽象的，并不是口腔中实际的舌位。所以“前后高低”更准确地说，只是描述元音之间的相互关系以及“听觉感知上的”音质。

# 音标的核心要点（DJ音标）（极简版全部音标）

# 元音

区分各个元音的决定性三要素是：

  1. “舌位”的前后
  2. “舌位”的高低
  3. 嘴唇的“圆唇度”

舌位可以理解为发音时舌面隆起的最高点在口腔中的位置。

感受一下舌位和圆唇度：

币：舌位前且高

兔：舌位后且高

哈：舌位中且低

这三个元音里只有兔是需要圆唇的，即圆唇度很高。

# /iː/

bee/biː/

舌位前、高，和拼音i基本一致。

ː 是长音符号，代表这个音标一般发音较长，但这不是绝对的。

还有个短版本，/i/，如happy/ˈhæpi/的尾音

# /ɪ/

big/bɪɡ/

舌位前、高，但是在/iː/的后下方（对比着学）。

在普通话里不存在。

特别注意，/ɪ/并不是/iː/的端版本/i/

# /uː/

too/tuː/

舌位后、高，高度和拼音u一致，但这个音在现代英语的舌位已向前迁移（看图），所以比拼音u更前。

和/iː/有短版本/i/相似，/uː/有个短版本/u/。比如influence/ˈɪnfluəns/中的第二个元音

# /ʊ/

book/bʊk/

舌位后、高，但在（朝前迁移的）/uː/的后下方，这个音在普通话里不存在

特别注意，/ʊ/并不是/uː/的端版本/u/

# /e/

bed/bed/

舌位前、中高，和普通话“夜”的韵母（拼音ye的e）一致

# /ɜː/

her/hɜː/

舌位前后、高低都在中央，有些母语人士发音可能低一点。听上去像拼音e，但实际上拼音e的舌位更后、更高一些。

比较：her/hɜː/、贺
