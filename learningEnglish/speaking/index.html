# 英语

先保证走完整个学习流程，然后再针对性地进行优化，提高效率节省时间。一个例子就是，先每天安排一定的量，即使当天没做完，第二天也不应该回过头去做第一天没做完的事，这种事应该留专门的时间去做，整个大方向不应该被这些小的突变影响，否则就是见了芝麻丢了西瓜。这个思维很重要，因为它保证了学习行为的持续性。而持续几乎是学习这件事里最重要的因素了，因为它是起决定性作用的，是所有矛盾的起点和根源。另一件几乎同等重要的事是纵深，如果说持续保证了量，那么纵深是保证了质。

# 音标

弄清楚每个最小单元（不可分），保证理解。然后再理解每个单元之间的关系，还可以通过这种方式去理解一些由基本单元形成的比较大的组合。

归根结底就是，不要让自己的感知始终处于自己的生活层面，而是要深入要所学的对象中去，要尽量丰富自己的感知，这样就能通过学习形成新的认知，就能解决自己感知不到除了自身之外的其他事物的问题。

比如说学英语的话，就拿学音标来说。如果你只是说学音标，背也背了，练也练了，但始终不用它去拼读单词，这样学习效果是不长久的，而你自身的认知也没有发生改变，整个音标对你来说就相当于一个黑箱，只是暂时被放到了你的知识体系当中而已，你甚至都没有理解它。但如果你用它去拼读单词，你看到单词的时候可能就会想这个单词用音标是怎么拼读的，这样，你知道音标是用来干嘛的，也知道它是怎么用的，那你就能够建立广泛的联系，你就是真正理解的它，而不是作为一个黑箱、一个你不理解的孤立的单元放在你的知识体系中了。所以，多层面感知很重要。

# 阅读

如果阅读的时候有一个很好的认知：词的功能（词性）、逻辑、词意，那么整篇文章就不再是独立的单词，而是有意义的，而这样的理解方式在复述的时候就比尝试孤立地理解单词然后记忆要好得多。注意，词意是排在最后的，有时你甚至不用搞明白词意，也能在词的功能和逻辑知道的前提下猜出来。
